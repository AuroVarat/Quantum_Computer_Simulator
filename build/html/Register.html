Register¶

Qbit Register Class initialises a quantum register with nqbits qbits and initialises all qbits to |0>. It contains method for qbits in the register.

class qRegister.QbitRegister(nqbits, cqbits=None, name='register')¶

Bases: qGates.QbitGate, qVisualiser.qVisualiser

Qbit Register Class initialises a quantum register with nqbits qbits and initialises all qbits to |0>l, and contains methods to apply gates to the qbits in the register.

measure_one_qubit(i_nqbit)¶

Measures the state of a single qbit in the quantum register. The qbit is specified by the index i_nqbits.

Parameters

  • optional) (i_nqbits (int,) – The Qubit to collapse. Required.

  • float – The collapsed state of the specified qbit.

output()¶

Prints the output of the quantum register probabilities in the basis space. Shows the states with the maximum probability. This method should be called after the measurement of the register.

Returns

basisSpace (list): The basis space of the quantum register.

sequence()¶

Prints the circuit sequence of the quantum register. Alternative to str() method.

set_state(state)¶

Sets the state of the quantum register to the state specified by the input state.

Param

state (int/list): can be a int showing the qubit state(“110”) or list of qubit states([ [1,0],[0,1],[0.5,0.5]]). The list supports float values.
